About this Project

This project is a part of Fantom Code Hackathon 2022 organized by RVITM. It is a bus ticket booking and verification app.

Verification is done using stickers and QR codes.

This project is open source. You can find the Github Repository here.

This project is a part of Fantom Code Hackathon 2022. It is made by Syed Ibrahim Maaz, Tejas Ganesh Joshi, Shravya and Harshita from RVITM.

Powered by GOQR API and Stipop API. Made with Bootstrap.
